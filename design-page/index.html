Input Statement

Input

An Input Statement reads a value from the keyboard and stores the result in a variable

Output Statement

Output

An Output Statement evaluates an expression and then displays the result to the screen


Declare Statement

Declare

A Declare Statement is used to create variables. These are used to store data while the program runs


Assign Statement

Assign

An Assign Statement calculates an expression and then stores the result in a variable

If Statement

If

An If Statement checks a Boolean expression then executes a true or false branch based on the result


While Loop

While

A While Loop evaluates a Boolean expression and, if true, executes statements. It rechecks the expression and loops until it is false


For Loop

For

A For Loop increments or decrements a variable through a range of values. This is a common and useful replacement for a While Loop
